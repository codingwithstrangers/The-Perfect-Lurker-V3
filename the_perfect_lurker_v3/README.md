# The-Perfect-Lurker-V3

## Chat Commands

### Host-only commands
- `!kick <username>` — Remove a viewer from the current race and add them to the ban list.
- `!unban <username>` — Remove a viewer from the ban list so they can join again.
- `!snapshot` — Save a timestamped stats snapshot for all tracked viewers.
- `!result` — Save/export the current race results.

### Viewer commands
- `!leave` — Leave the current race (uses your own username automatically).
- `!place` — Show your current race placement.
- `!defense` — Show your defense stats.
- `!rival` — Show your rival stats.
